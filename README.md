기능 설명

1. 로그인 페이지에서 로그인하여 랜딩페이지로 이동
   1-1. 로그인 페이지에서 가입하기 버튼 클릭하여 회원가입 후 로그인

2. 랜딩페이지의 유저아이콘 -프로필 클릭하여 유저페이지로 이동

3. 랜딩페이지의 유저아이콘 - 설정 클릭하여 프로필 사진 설정
